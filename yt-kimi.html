📋 API Information

This application is powered by three robust APIs working in harmony:

  • BK9 API — Primary video metadata and download link provider. Returns filename, quality, size, and direct MP4 URLs.
  • EliteProTech API — Secondary fallback for video downloads. Provides title and download URLs with MP4 format support.
  • NexRay API — Thumbnail extraction, duration, resolution details, and audio (MP3/FLAC) download links. Also handles video downloads with quality selection.

The system intelligently routes requests across these APIs for optimal performance and reliability.
